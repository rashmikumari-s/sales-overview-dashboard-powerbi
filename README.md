# Sales Overview Dashboard (Power BI)

This project is a Power BI dashboard built to analyze sales data and provide insights using interactive visuals.

## Features
- Sales by Region
- Sales by Segment
- Sales by Category
- KPIs for Total Sales and Total Profit
- Slicers for dynamic filtering

## Tools Used
- Power BI
- Excel

## Purpose
This project was created to practice dashboard design, data visualization, and business reporting skills using Power BI.# sales-overview-dashboard-powerbi
